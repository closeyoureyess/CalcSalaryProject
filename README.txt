#Инструкция по запуску проекта

## 1. Скачайте проект

##2. Откройте проект в IntelliJ IDEA
Запустите IntelliJ IDEA.
Перейдите в меню File и выберите Open....
Найдите и выберите директорию с проектом. Нажмите OK.

##3. Подождите, пока IDEA завершит индексацию
IntelliJ IDEA автоматически загрузит все необходимые зависимости и выполнит индексацию проекта.

##4. Настройте JDK
Убедитесь, что проект использует JDK 11.
Перейдите в меню File -> Project Structure.
В разделе Project выберите Project SDK и установите JDK 11, если он еще не установлен. (Рекомендуется Amazon Coretto)
В разделе Modules убедитесь, что Module SDK установлен на JDK 11.

##5. Запустите проект
Найдите класс CalcSalaryProjectApplication.
Нажмите правой кнопкой мыши на класс и выберите Run 'CalcSalaryProjectApplication.main()'
Убедитесь, что проект успешно запустился

##6. Откройте Swagger-документацию
После успешного запуска приложения откройте веб-браузер и перейдите по адресу:
http://localhost:8080/v3/api-docs - по адресу находится документация для API в формате Swagger
